Fingers Shortcuts

   A computer vision application with implementation of AI smart hand detection, which allows users to create specific operation and assign them to fingers pattern, performing those tasks when camera detects the specified fingers pattern.

Application Features

Smart Detection

Can detect hand landmarks wherever the hand position.

Fast Performance

Performing shortcuts fast once they have detected.

Multiple Options

Provides lots of user options and specifications.
- OpenCV (Open Source Computer Vision) is an open source computer vision and machine learning software library.
- Python is an interpreted high-level general-purpose programming language.
- NumPy is an open source project aiming to enable numerical computing with Python.

Application Screenshots

Application main window, contains list of user option including camera options used in webcam connection and its port number, in addition to visualization options allowing users to define the visual screen size, view statistics, and color of visual screen text, etc.

Shortcuts tab, responsible for shortcuts management: adding, editing, deleteing or clear the whole data.
Shortcuts can be either opening file\s or pressing keyboard keys sequence or performing specific command using CMD.

Visual window, appears when visual option (in display tab) is checked showing the hands landmarks detection and the pattern of raised fingers.
